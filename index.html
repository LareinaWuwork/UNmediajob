Built for UN communicators.

This project helps organize media, communications, journalism, video, social media, public information and external relations opportunities across the United Nations system. The current version is a clean public website. The next version can connect to GitHub Actions and Python to update listings automatically every day.

How it works

A simple workflow: collect roles, filter by communications keywords, publish them online, and update regularly.

1

Collect

Search UN career pages for media, communications and public information roles.

2

Filter

Keep only roles related to journalism, digital content, video and external relations.

3

Publish

Display selected roles through a clean GitHub Pages website.
